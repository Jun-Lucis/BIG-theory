# BIG-B9 Figures

This directory collects representative figures for **BIG-B9**, the reduced boundary-energy model for fission-like metastability.

The figures here are intended for visual orientation and documentation. They should be read together with the B9 paper notes, Zenodo records, and the limitation statements in the main repository.

---

## Scope

BIG-B9 studies the reduced energy

$$
E(\Omega;\lambda)=\sigma P(\Omega)+\lambda C(\Omega),
$$

where (P(\Omega)) is a boundary-cost term and (C(\Omega)) is a Coulomb-like nonlocal repulsion term.

The main visual theme is the emergence of a fission-like metastable structure:

```text id="4aevii"
compact state
    -> finite pinch barrier
    -> separated branch
```

The figures in this folder should be interpreted as structural and model-level illustrations, not as quantitative nuclear-fission predictions.

## Important caution

The B9 figures should not be presented as calibrated nuclear-fission plots.

In particular:

* (\lambda) is not calibrated to MeV;
* (x_{\mathrm{proxy}}=Z^2/(50A)) is only a coarse fissility-related proxy;
* barrier values, if shown, are representative benchmark indicators;
* no quantitative fit from BIG-B9 to empirical fission barriers is claimed.

Preferred wording:

```text id="y2grfy"
macroscopic structural comparison
```

```text id="k3ulnk"
fission-like metastability in a reduced boundary-energy model
```

Avoid unqualified wording such as:

```text id="4lkgiz"
B9 predicts nuclear fission barriers
```

```text id="9zzvgy"
B9 quantitatively fits actinide fission data
```
